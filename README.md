# Pop-and-Relax 🫧

A minimalist and relaxing bubble-popping experience designed for quiet moments and self-care.

✨ Features
* **Satisfying Interactions:** Simple bubble-popping mechanics.
* **Haptic Feedback:** Includes vibration for a more tactile feel.
* **Aesthetic Sound Design:** Uses a custom `SoundPool` for relaxing audio effects.
* **Clean UI:** Indie-sleaze inspired minimalist design.

✨ Installation
1. Go to the **Releases** section on the right side of this page.
2. Download the `app-debug.apk`.
3. Open the file on your Android device and install (allow "Install from unknown sources" if prompted).
